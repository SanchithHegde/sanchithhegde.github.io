Uses

Editor + Terminal#

I use Konsole that ships with the KDE Plasma desktop environment with zsh along with zinit plugin manager and powerlevel10k theme on my Arch Linux system. I use VS Code with the Ayu Dark theme for most of my development work. The only exception is when I’m working on something Android related, for which I use Android Studio. I also tend to use micro for quick edits on the terminal.

All my config files can be found in my dotfiles repo.
